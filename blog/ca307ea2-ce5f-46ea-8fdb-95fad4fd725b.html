判断两个矩形相交

这篇文章发表于 2022年05月17日，星期二，21:43

在画图工具中需要判断两个矩形是否相交

如上有两个矩形，中心点分别为O1(x1、y1)、O2(x2、y2)，宽高分别为(w1、h1)和(w2、h2)；

要判断两个矩形是否相交，可以判断矩形的两个中心点的水平垂直距离是否小于(w1 + w2)/2、(h1 + h2)/2。

function cross() {
	return Math.abs(x2 - x1) < (w1 + w2)/2 && Math.abs(y2 - y1) < (h1 + h2)/2;
}
