mac窗口的插件vwin-eazy-win

这篇文章发表于 2021年12月17日，星期五，21:54

以前看很多博客文章，都用mac的窗口样式把图片或者文章包裹起来，话空闲时间自己也做了一个

效果如下：

安装：

npm install -S vwin-eazy-win

使用方式:

import EazyWin from 'vwin-eazy-win';

function Demo() {
return <EazyWin>
		<img {...reset} loading="lazy" />
	</EazyWin>
}

属性：

名称默认值
className''
themedefault，可选深色模式dark
